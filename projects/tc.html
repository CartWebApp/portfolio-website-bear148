Tank Clicker

Technologies Used

Project Description

“Tank Clicker” was a personal project that I spent a lot of time developing. It is a game where you click on tanks, buy upgrades, and upgrade your own tank following a tech tree. The game is based around World of Tanks. The vast majority of development was spent on JavaScript. The HTML structuring and CSS was all pretty straightforward, but this project relied heavily on my JavaScript skills to work properly. The game automatically saves user data every time they close the window or tab, you can purchase different upgrades with randomizing icons to create variety, you can upgrade your tank going up 1 tier every time you upgrade, following the same tech tree as used in World of Tanks.
